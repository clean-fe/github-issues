# clean-fe-template

### 설치순서

```shell
git clone https://github.com/clean-fe/github-issues.git

cd github-issues

npm install

npm run dev
```

## 2주차: 객체지향  프로그래밍

### 학습 계획

- 도서 <오브젝트> 중요한 부분 복습
- TS를 사용하여 도서 <오브젝트>의 내용 적용시켜보기
- data binding 기술 개념
  - ES Classes 의 setter, getter 
  - Proxy API 사용 
  - defineProperty 를 활용 
  - Observer 패턴
- decorator
